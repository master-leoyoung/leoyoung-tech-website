# Enterprise Edition

While TeamMind's open-source free version can meet the needs of most small and medium-sized users, and users with development capabilities can even build their own features upon it; 

The Enterprise Edition is specifically designed for large organizations or those with special requirements. Compared to the open-source free version, the Enterprise Edition offers a professional management backend "**TeamMind Web Admin**" , whose main features include:

- **Comprehensive User Management:** Supports one-time import of all users, pre-creation of group hierarchies, and individual user management.
- **Enterprise Knowledge Base Management:** Creation and management of various knowledge bases, assigned to different groups.
- **Enterprise AI Tools Management:** Creation and management of various Skills and MCPs (Model Context Protocols), assigned to different groups.
- **Full User Dialogue Management:** Supports query auditing, statistical analysis, and intent analysis of all user dialogues.
- **SiteMind Module Integration:** Ability to load the SiteMind functional module.

**Additional Benefits:**

- Provides 5*8 one-on-one email support services.
- Offers appropriate OEM and customization options.
