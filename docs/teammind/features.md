# Features

TeamMind is built around the group as its fundamental unit, embedding AI capabilities into every layer of how teams work together.

## 🤝 Group-Native AI Collaboration

Multiple members work with AI together inside the same group — not in parallel silos, but as a unified team.

AI is no longer anyone's private assistant. It's shared intelligence of the whole group.

**Multiple groups, multiple members** — flexible structure that mirrors how organizations actually work.

Each group maintains its own independent AI context and conversation history.

Members can communicate with each other using `@` mentions; AI listens and records, but does not interrupt person-to-person conversations.

Switch seamlessly between TeamMind AI and any connected third-party LLM within the same group.

## 🧠 Persistent Memory & Knowledge Compounding

This is what sets TeamMind apart from every comparable tool.

Every group conversation is permanently recorded and automatically structured into three layers of knowledge:

| Knowledge Base Type         | Description                                                                                                                |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **Document Knowledge Base** | Static documents uploaded by users — foundational layer of group knowledge                                                 |
| **Group Collection Base**   | Key insights automatically extracted from conversations, structured into summaries                                         |
| **Dialogue Record Base**    | Full conversation archive with RAG retrieval — captures hidden reasoning, brainstorming, and evolving context in real time |

**Key Points**: TeamMind automatically distills the most important knowledge in each group into a browsable summary. New members get up to speed instantly — no need to read through thousands of messages.

**Correctable Learning**: When TeamMind gives a wrong answer, click 👎, provide the correct response, and it won't make the same mistake again.

## 🔒 Group Isolation: A Natural Security Boundary

Every group's knowledge base and AI tools are fully independent from all other groups — by design.

**RAG is group-scoped**: A group's knowledge base is not accessible to other groups by default.

**Authorized sharing**: Administrators can explicitly share a knowledge base with other groups, building a cross-group "knowledge value chain" without duplication.

**Tiered access control**: Group administrators have full control over member permissions, knowledge base access, and AI tool availability.

## 🖥️ Flexible Deployment: Your Data, Your Rules

TeamMind offers maximum deployment flexibility to meet the needs of organizations of any size and security posture:

| Deployment Mode            | Description                                                                              |
| -------------------------- | ---------------------------------------------------------------------------------------- |
| **SaaS (Cloud)**           | Ready to use immediately — no infrastructure required                                    |
| **On-Premise**             | All data stays entirely within your own environment                                      |
| **Custom LLM API**         | Connect your preferred or proprietary large language model                               |
| **Bundled LLM (Optional)** | On-premise deployments can include a pre-configured LLM — no external model setup needed |

Regardless of deployment mode, the full TeamMind experience remains identical.

## 🌐 SiteMind™: Your Website as a Community Gateway

Turn your website into the starting point of an AI-powered community.

Visitors can join your TeamMind group with a single click — right from any page.

**Three steps to deploy:**

1. Embed SiteMind™ component on any page of your website
2. Visitors chat with AI, which answers product questions accurately using RAG
3. Interested visitors register and join your TeamMind group directly

Not a pop-up ad. A genuinely useful conversation.

Turns one-time traffic into a continuously growing, engaged community.

SiteMind™ is available in the Commercial Edition.

## 📄 Advanced Document Recognition

The value of a knowledge base depends on how much it can actually *understand*.

TeamMind has invested deeply in document processing — especially for complex, real-world document formats enterprises actually use:

**Image Content Recognition**: Understands the content of image-based documents, not just extracting embedded text.

**Mixed Layout Recognition**: Correctly interprets documents where text and images are interwoven — product manuals, research reports, technical specs — preserving the semantic meaning of layout.

**Zero Preprocessing Required**: Upload directly. TeamMind handles parsing automatically, with no manual format conversion needed.

**Broad Format Support**: `PDF` · `Word` · `PPT` · `Excel` · `CSV` · `TXT` · `HTML` and more.

## 🔧 MCP & Skill Extensions

For professional teams with advanced customization needs, the Commercial Edition provides:

**MCP (Model Context Protocol)**: Connect group AI to external tools and live data sources, extending far beyond the knowledge base.

**Custom Skills**: Configure dedicated AI skills and automated workflows for specific groups and use cases.

The base edition includes full RAG knowledge base capabilities — sufficient for day-to-day needs of most teams.
